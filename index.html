Usage

  1. Install tool
sudo pip install docutils pygments
  1. Download
git clone  https://github.com/Rykka/rhythm.css.git
cd rhythm.css
  1. Generate Html
rst2html.py /path/to/your/rst \
     --stylesheet=rhythm.css,syntax/molokai.css \
     --syntax-highlight=short \
     > /path/to/the/output_html
